# MCP Setup for Finova

This repository is prepared for integration with Model Context Protocol (MCP) servers.

## Available Integrations
- **GitHub MCP**: Use `mcp/github` for custom GitHub actions and PR interactions.
- **Firebase MCP**: Use `mcp/firebase` for database migrations and rule testing.
- **Chrome DevTools MCP**: Use `mcp/chrome-devtools` for automated browser testing and UI verification.

## Getting Started
Each folder in `mcp/` contains specific configuration for its respective MCP server.
To connect to an MCP server, ensure you have the appropriate MCP client configured with your AI assistant.
